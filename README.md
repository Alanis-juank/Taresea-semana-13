Esta herramienta fue creada con Python empleando la biblioteca Tkinter para establecer una interfaz gráfica de usuario (GUI).
La finalidad del software es permitir que el usuario introduzca datos a través de un cuadro de texto y pueda ver esa información en una lista dentro de la interfaz.

La aplicación reacciona a las acciones del usuario, como los clics en los botones, lo que facilita la adición y eliminación de datos de forma sencilla.

---

# Diseño de la Interfaz

La interfaz gráfica incluye los siguientes elementos:

- Etiqueta: Presenta el texto "Ingrese un dato".
- Campo de texto: Ofrece al usuario la posibilidad de escribir datos.
- Botón "Agregar": Sirve para incorporar el texto que se ha escrito en la lista.
- Lista (Listbox): Enseña los datos que el usuario ha añadido.
- Botón "Limpiar": Suprime todos los contenidos de la lista.

La interfaz fue creada con el objetivo de ser sencilla, comprensible y fácil de operar.

---

# Funcionalidades

La aplicación tiene la capacidad de:

- Recibir datos en un campo de texto.
- Añadir información a una lista mediante un botón.
- Mostrar los datos ingresados en la interfaz.
- Vaciar la lista utilizando el botón "Limpiar".

---

# Eventos

La aplicación responde a las acciones del usuario:

- Clic en el botón "Agregar": activa una función que toma el texto introducido y lo añade a la lista.
- Clic en el botón "Limpiar": elimina todos los elementos presentes en la lista.

# Tecnologías utilizadas

- Python
